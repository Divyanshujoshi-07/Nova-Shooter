# Nova-Shooter 🚀


Nova-Shooter is a futuristic arcade-style space shooter game built using HTML, CSS, and JavaScript. Inspired by classic retro shooters with a modern cyberpunk aesthetic, the game delivers fast-paced action, glowing neon visuals, smooth animations, and immersive gameplay.

Players control a powerful spaceship and battle through endless waves of enemies while dodging attacks, collecting power-ups, and surviving intense combat scenarios. The game combines retro arcade mechanics with modern UI effects to create an exciting browser-based gaming experience.

## ✨ Features

- 🎮 Smooth and responsive spaceship controls
- 👾 Dynamic enemy wave system
- ⚡ Power-ups and weapon upgrades
- 💥 Explosive visual effects and animations
- 🌌 Cyberpunk neon-themed design
- 🔊 Arcade-style gameplay experience
- 📱 Responsive design for different screen sizes
- 🚀 Built completely with HTML, CSS, and JavaScript

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla JS)

## 🎯 Gameplay

Control your spaceship, destroy enemy fleets, avoid collisions, and survive for the highest score possible. As the game progresses, enemies become faster and more challenging, testing your reflexes and strategy.

## 📂 Project Structure

- `index.html` – Main game structure
- `style.css` – Game styling and neon effects
- `script.js` – Core game logic and mechanics
